# DnD-5e-5th-Edition-Spell-Card-Generator

# Overview

This project automates the generation of the following for two distinct product lines:
  - A database of 5th Edition DnD spell information.
  - A database of 5e format DnD spell information.

The script used for this generates .csv files for each database intended for wider use in a digital asset automated production pipeline. 

The digital asset automation has been omitted from this repository as the assets produced are for commercial use. 

# Scope

This repository contains:
  - The script that generates and transforms the data which is outputted as a .csv file.
  - Details of the project development (testing, problems, solutions etc.)
  - Evidence of the asset generation portion of the pipeline.
  - Evidence of the final production pipeline results.
